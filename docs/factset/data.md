# Available Data

FactSet provides a wide range of financial data and analytics:

- **Company fundamentals** — financial statements, ratios, and estimates for public companies worldwide
- **Equity and fixed income pricing** — historical and real-time price data across global markets
- **Ownership data** — institutional and insider holdings, fund flows
- **Industry classifications** — FactSet RBICS (Revere Business Industry Classification System) for sector and industry screening
- **Screening and charting** — built-in tools for filtering companies by financial criteria and creating visualizations
- **Excel plugin** — pull FactSet data directly into Excel spreadsheets using FactSet formulas and the sidebar
