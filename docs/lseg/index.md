# LSEG Workspace

!!! warning "Access Currently Not Available"
    LSEG Workspace (formerly Refinitiv Eikon / Thomson Reuters Eikon) is **not currently available** at Utrecht University. This page will be updated if access becomes available in the future.

## About LSEG Workspace

LSEG Workspace is a financial data and analytics platform provided by the London Stock Exchange Group. It offers real-time and historical data on equities, fixed income, commodities, foreign exchange, and more, along with news, analytics, and screening tools.

If you have questions about data access, contact **financedata.use@uu.nl**.
