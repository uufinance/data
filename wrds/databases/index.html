    Available Data¶

    The following databases are available to Utrecht University via WRDS. Click the guide link for details on available tables, API library names, and what data you can expect to find.

    For a full overview of all variables and tables, see the WRDS Data Dictionary.

    Corporate Financials¶

    Database Description WRDS Guide
    Compustat North America Corporate financial statements and stock prices for US and Canadian companies Link Guide
    Compustat Bank Fundamental financial data for North American banks Link Guide
    Compustat Global Global company fundamentals, stock prices, exchange rates, and index data (80+ countries) Link Guide
    Compustat Historical Segments Business and geographic segment-level data (revenue, assets by segment) Link Guide

    Compensation & Governance¶

    Database Description WRDS Guide
    ExecuComp Executive and director compensation data for S&P 1500 companies Link Guide
    Audit Analytics Audit fees, audit opinions, auditor changes, financial restatements, and SOX compliance data for SEC registrants Link Guide

    Fixed Income & Bond Markets¶

    Database Description WRDS Guide
    Mergent FISD Fixed income securities: bond issue details, issuer information, credit ratings Link Guide
    TRACE US corporate bond and agency debt transaction data Link Guide
    Bond Returns Monthly corporate bond returns constructed from TRACE Link Guide

    Market Data & Indices¶

    Database Description WRDS Guide
    World Indices Daily and monthly price data for global stock market indices Link Guide
    S&P Dow Jones Indices Historical index levels, returns, and constituents for S&P and Dow Jones indices Link Guide
    Fama-French Portfolios and Factors Fama-French factor returns (3-factor, 5-factor, momentum), research portfolios, industry classifications, and liquidity factors Link Guide
    OTC Markets Securities traded on OTC markets (OTCQX, OTCQB, Pink) Link Guide
